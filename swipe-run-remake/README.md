# 3レーン・スワイプラン

スマホ縦画面向けの3レーン・スワイプランゲームです。既存の `o-kyan 2/index.html` とは別プロジェクトとして動きます。

## 遊び方

- 左右スワイプ、または左下の左右ボタンでレーン移動
- タップ、右下の JUMP ボタン、または Space / ArrowUp でジャンプ
- 案内テーブルやゲートを避ける
- 丸い穴カバーはジャンプで回避
- コインを集めるとリザルトでスコアに換算
- コインを連続で集めるとコンボ倍率が上がる
- コイン5枚以上を持っていると、1回だけレスキューで復帰
- 青い床はスピードアップ、黄色の注意マットはスピードダウン

## 公開

ビルド不要です。GitHub Pages ではこのフォルダの `index.html` を公開対象にしてください。

発表時は `presentation.html` を開くと、スマホで読み取るためのQRコードと操作説明を大きく表示できます。
